# ICGroup Backend

Backend-часть проекта ICGroup.  
Здесь будет разрабатываться серверная логика, API и работа с базой данных.

## Технологии

- Node.js
- TypeScript
- (позже) Express / NestJS / Fastify
- (позже) Prisma / PostgreSQL

## Назначение

Этот репозиторий будет содержать:

- API для фронта и админки
- авторизацию и роли
- работу с базой данных
- бизнес-логику проекта
